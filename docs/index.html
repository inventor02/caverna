Caverna

GitHub | Issues | Documentation

Introduction

Caverna is a rogue-like game written in C, using OpenGL for graphics. It's currently in development.

Project Team

Caverna is developed by four students in ECS at the University of Southampton - George Peppard, Toby Godfrey, Adam Caine and Ellie Heywood ({gjp1g21, tmag1g21, ac3g21, ejh3g21}[at]soton.ac.uk). We're working on this project because we have some (not much!) spare time and wanted a fairly long-term project to work on in between lectures.

Documentation and Issue Tracking

We use Atlassian Jira for issue tracking and document code style and other information for both contributors and users on Atlassian Confluence. Links to both of these platforms are available at the top of the page and anyone may register for an account.

Source Code

Caverna is open source and we welcome contributions. You can find us on GitHub using the links at the top of the page. Please take some time to read the documentation before contributing. The project is licensed under the GNU General Public License 3.0.

Releases

Caverna is not currently available for release. We will update this page when working builds are available.
